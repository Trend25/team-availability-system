GitHub Repository Kurulumu

Bu sistem GitHub repository'sini bulut veritabanı olarak kullanır. Tamamen ücretsiz ve güvenilir.

Adım 1: GitHub Repository Oluşturun

  1. GitHub'a gidin: github.com
  2. "New repository" butonuna tıklayın
  3. Repository adı: team-availability-data
  4. "Public" seçin (özel veriler için "Private" da olabilir)
  5. "Add a README file" işaretleyin
  6. "Create repository" tıklayın

Adım 2: GitHub Personal Access Token

  1. GitHub Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. "Generate new token" (classic)
  3. Note: "Team Availability System"
  4. Expiration: "No expiration" veya "1 year"
  5. Scopes: "repo" işaretleyin
  6. "Generate token" ve kopyalayın
Önemli: Token'ı güvenli bir yerde saklayın. Bir daha gösterilmeyecek!
